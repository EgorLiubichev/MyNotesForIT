[cls] - очистить командную строку.
[md] - (make directory) создать директорию.
[dir] - show folders and files in the directory.
[cd название папки] - change directory
[cd ..] - перейти к родительской папке.
[mkdir folder_name] - создать папку в данной директории (make directory).
Tab может дописывать возможное из имеющихся название папки или файла, если написать первую букву.
	Пример: cd f + [tab] = cd folder
[hostname] - команда отображает имя компа, на котором мы находимся
[tasklist] - выводит список всех процессов
[taskkill] - убивает процесс. Но нужно передать дополнительные параметры (process ID и другое) - [taskkill /?] - поможет узнать необходимые параметры и примеры.
[ipconfig /all] - список сетевых интерфейсов
[systeminfo] - отображает свойства системы
[robocopy] - команда для копирования файлов
