Batch файлы пишутся в обычный блокнот. Можно использовать любые команды, используемые в CMD. Файл нужно сохранить с расширением .cmd

Пример:
title IP Config script
echo Starting ipconfig...
ipconfig /all > result.txt
echo Done! Check result.txt
pause

В первой строчке изменен title окна
Во второй - то, что пишется для пользователя (это он увидит в открытой консоли)
В третьей: запуск самой команды
Четвертая: опять для пользователя
Пятая: пауза ставится для того, чтобы программа не закрылась моментально после того, как отработает

Для того, чтобы скрипт работал в командной строке как полноценная программа, нужно в переменных среды в Path дописать ту директорию, где лежит файл. Т.к. ком.строка ищет все имеющиеся программы в тех директориях, кот. прописаны в path.
