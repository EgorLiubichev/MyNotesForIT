Зажатый Alt часто в приложении позволяет увидеть горячие клавиши.
F1 вызывает справку практически во всех приложениях.

Browser:
Ctrl + t - новая вкладка (страница)
Ctrl + n - новое окно
Ctrl + Tab - переключение м/у вкладками.
Ctrl + Shift + Tab - переключение м/у вкладками в обратном порядке.
Ctrl + w - закрыть вкладку.
Ctrl + L - выделяет поле URL (для копирования или замены, или др.)
Ctrl + F - поиск на странице.
F11 - FullScrean режим.
F12 - Developer tools (для работы с ФронтЭндом).

Windows:
Win + p - работа с мониторами
Win + m - свернуть все окна
Ctrl + Shift + Esc - task manager (диспетчер задач)

IDE:
Ctrl + D - дублирует строчку кода.
Ctrl + Y - удаляет строчку кода.
Shift + Home - выделить строчку кода.
Shift + End - выделить строчку кода, если курсор в начале строки.
Alt + Insert - открывает список с конструктором, геттероми, ту стрингом и пр.
Shift  и стрелки позволяют выделять для выбора поля в списке выбираемых.

Горячие клавиши можно подсмотреть на сайтах разроюотчиков приложений ;)
